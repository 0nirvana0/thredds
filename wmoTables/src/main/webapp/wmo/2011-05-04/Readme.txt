3. Notes to entries are available in separate files on the WMO server. URL base to the relevant files are;

http://www.wmo.int/pages/prog/www/WMOCodes/WMO306_vI2/2010edition/...../

"....." is nine charcters and differs from table to table; i.e. BUFRver16 (for BUFR/CREX version 16), GRIB2ver7 (for GRIB2 version 7) and CCT_May11 (for Common Code tables).


If the file name/bookmark in Note_E column, for example, is "BUFR_16_0_0_tableD.doc#B_Cat01", full URL to the Note is;

http://www.wmo.int/pages/prog/www/WMOCodes/WMO306_vI2/2010edition/BUFRver16/BUFR_16_0_0_tableD.doc#B_Cat01.
