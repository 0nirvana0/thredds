Upgrading To TDS 4.6

New Requirement

  • Java 7 is required
  • Tomcat version 7.0.43 or greater

    •  

GRIB Feature Collection Changes

  1. New indexes (.ncx3) will be written the first time GRIB files are accessed. You may delete the older indices (ncx, ncx2) if you like. The gbx9 indices can stay, althoug deleting them will allow a few minor improvements to be made.

For complete information, see GRIB Collections and TDM.

 

New Features in version 4.6

API changes

Client Catalog API

The catalog processing classes in thredds.catalog package has been replaced by thredds.client.catalog everywhere on the client. If you are using thredds.catalog in your code, please switch to thredds.catalog.client, as it is takes less memory and is better performant. The old package is still available in the compatibility.jar, which you will need to sperately include in your classpath if needed.
