NetCDF Java

The NetCDF Java library implements the Common Data Model (CDM) to interface netCDF files and other types of scientific data formats.

See the netCDF Java package overview
Other NetCDF Libraries
  • NetCDF supports the creation and access of array-oriented data.
  • The use of LibCF allows data files to conform to CF conventions.
Common Data Model

The NetCDF-Java library implements the Common Data Model (CDM), an abstract data model. The CDM merges the netCDF, OPeNDAP, and HDF5 data models to create a common API for many types of scientific data.

 netcdfgroup is an unmoderated mailing list for announcements, discussions, bug reports, and questions about netCDF software.
